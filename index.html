📱 新增到主畫面
Step 1 · 上傳到 GitHub
① 在 github.com 建立新的 Repository
② 上傳這個 index.html 檔案
③ 進入 Settings → Pages → 選擇 main branch → Save
Step 2 · 在 iPhone 加入書籤
① 用 Safari 開啟你的 GitHub Pages 網址
② 點底部的 分享按鈕 ⬆️
③ 選「加入主畫面」
④ 自訂名稱後點「加入」
✨ 書籤圖示會使用你在外觀設定中選擇的 Emoji 圖示，可隨時更換。
